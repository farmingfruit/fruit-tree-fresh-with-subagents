Testing Instructions

  1. Open the browser console to see measurement logs
  2. Start with a wide browser window - all navigation items should be visible
  3. Slowly resize the window narrower
  4. Watch for when the "Events" tab disappears and a "More" button appears
  5. Click the More button to see the dropdown with Events
  6. Navigate to Events through the dropdown - notice the active state
  7. Click outside the dropdown to close it
  8. Continue resizing to test edge cases
